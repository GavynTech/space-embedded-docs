# Latency & Timing Numbers

The numbers every space systems engineer should have internalized. Not for memorization — for intuition about what your code is doing and what is physically possible.

## Software Timing Budget (10 Hz ADCS Loop, 100 ms period)

| Step | Time Budget | Notes |
|---|---|---|
| IMU read (SPI) | ~10 µs | 3-byte register read at 10 MHz SPI |
| Magnetometer read (I2C) | ~500 µs | 6-byte at 400 kHz I2C |
| EKF propagation | ~500 µs | Matrix operations on Cortex-M7 |
| EKF measurement update | ~1 ms | If reference available |
| PD control law | ~50 µs | Simple multiplication |
| Reaction wheel command (SPI) | ~10 µs | |
| Telemetry packing | ~100 µs | |
| **Total** | **~2.2 ms** | **22 ms remaining as margin** |

78% CPU headroom at 10 Hz on a typical embedded processor. This headroom absorbs measurement update computation, additional sensors, and fault checking without threatening the deadline.
